¿Qué servicios ofrecemos?

Tipos de sensores:


Sensor de temperatura El sensor evalua, mide y recopila la temperatura ambiental, ajustando el tiempo entre cada medición. Efectividad de entre -15 y 60 ºC.

Sensor de humedad Este sensor realiza un control a tiempo real sobre la humedad, permitiendo así regular el riego de manera más eficiente.

Sensor de salinidad El sensor está diseñado para controlar el nivel de salinidad de los pozos de riego de las diferentes plantaciones que se pueden poseer.

Sensor de iluminación Este sensor controla los niveles de luminosidad, para controlar tanto las situaciones climatológicas como los invernaderos.

Acelerómetro El sensor está diseñado para detectar vibraciones que le afectan a la sonda, en función del tiempo, para comprobar si lo roban.
